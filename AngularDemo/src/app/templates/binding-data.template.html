Binding can be one way TO the template (data-binding), one way FROM the template (event), or two-way binding (models/forms), which we will see later when we get to forms.

  • [] surround a one way FROM the component to the template.
  • () surround an event raised on the template that calls ANY typescript.
  • [(ngModel)] is a special call that us used for two-way binding. This is a "Special sauce" tag.

One-way binding can be bound to input, and that input can be updated over time. Speaking of time, here is the current time: {{currentTime | date:'mediumTime'}}

Event binding is a way of sending information from an "inner" scoped source to an "outer scoped" target. For example, we can take a text input field and capture that in a property.
