Components can be really heavy. For a lighter touch, directives are used. A directive attaches a "process" to any element. Directives can have @Input parameters as well.
Its good practice to use directives instead of components whenever possible.

A directive can also add all sorts of business functionality. For example, what if we needed the first character of every input to be upper case. Or, what if we need a user input that only allows for numbers? A directive would be a perfect way to achieve this.
